# Prototype Returns & Refunds Policy (Synthetic)

This document is **synthetic** for the Mumzworld AI Intern take-home prototype.
It is **NOT** Mumzworld’s real policy.

## Eligibility windows
- Most items: return request within **14 days** of delivery.
- Damaged/defective items on arrival: report within **48 hours** of delivery and provide clear photos.
- Wrong item shipped: report within **7 days**.

## Condition requirements
- Items must be **unused**, **unwashed**, and in the **original packaging** with tags/labels.
- Bundles must be returned **complete** unless an item is missing/defective.

## Non-returnable categories (hygiene & safety)
- Opened consumables (e.g., baby formula), opened diapers/wipes, breast pumps after opening/usage, underwear, swimwear, and personal-care items **once opened** are not eligible for refund or exchange.
- If a non-returnable item is **defective on arrival**, escalate for manual review.

## Resolution guidelines
- Damaged on arrival + photos: **exchange or refund** (customer preference) if stock exists.
- Wrong item shipped: **exchange**; refund if the correct item is unavailable.
- Change of mind / wrong size (resellable): **exchange or store credit**. Refund only if within window and item is resellable.
- Late requests outside the window: **store credit or refuse** unless exceptional.

## What we need from the customer
- Order number, item name/SKU, delivery date, reason, and photos for damage/defect.

## Uncertainty rules
- If key information is missing, ask follow-up questions instead of guessing.
- If the request is outside policy or the intent is unclear, set low confidence and escalate.
